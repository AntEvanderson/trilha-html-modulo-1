Surgimento da Internet

Voltar

Primeiramente, entendemos a história da web, começando pelos primeiros computadores em 1944 até os tempos atuais e suas gerações.
Discutimos como surgiu a internet, mencionando a época da Guerra Fria na década de 1950. A primeira conexão foi estabelecida em 1969
pela UCLA (University of California) e pelo SRI (Stanford Research Institute). A primeira mensagem enviada foi a palavra "LOGIN", mas apenas "LO" chegou ao destino.

Explanando HTML

Voltar

O HTML (HyperText Markup Language) é uma linguagem de marcação amplamente utilizada para a construção de websites.
Sua estrutura é baseada em textos e caixas de marcação, formando o que muitos chamam de HTML "cru", sem design ou animações.

Tags HTML

Voltar

Conhecendo as estruturas básicas do HTML, incluindo a abertura e fechamento de tags, cabeçalho (head), corpo (body) e título (title). Também abordamos os atributos input type para criar campos de texto
e vários atributos para modificar texto: itálico (i), negrito (strong), sublinhado (u) e marcação (mark). Para textos, aprendemos algumas tags de cabeçalhos com 6 níveis de hierarquia (h1 até h6)
, para parágrafos (p), e para usar citações (blockquote). Para listas ordenadas e não ordenadas, usamos as tags ol (ordenadas) e ul (não ordenadas), e para adicionar itens às listas usamos (li).
Para adicionar links às suas páginas web, usamos (a href="link"), com características como (target="_blank") para abrir o link em outra aba e (title="texto") para exibir uma espécie de balão de texto
ao passar o mouse sobre o link.
